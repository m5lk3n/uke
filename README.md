# uke

**this fun project is work in progress!**

Print basic Ukelele chords like so e.g.:

```
C
+==+==+==+
|  |  |  |
+--+--+--+
|  |  |  |
+--+--+--+
|  |  |  3
+--+--+--+
|  |  |  |
+--+--+--+
1 = index finger, 2 = middle finger, 3 = ring finger, 4 = pinky
```

The plan is to make this available via an API.

## basic mechanics

``` test
$ go test
```

``` run

```
